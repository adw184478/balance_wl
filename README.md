# balance_wl
基于英飞凌 TC264 主控开发轮腿一体化机器人，完成硬件底层驱动、姿态滤波、平衡串级 PID、腿部运动学逆解与视觉循迹全套软件。采用一阶互补滤波抑制陀螺仪漂移，视觉大津法 + 八邻域算法识别赛道，实现自平衡、姿态调节、循迹、越障跳跃功能，适配电子设计竞赛场景。
